# Python Basics for Beginners

A beginner-friendly Python learning repository covering the core fundamentals of Python programming with simple examples and explanations.

## Topics Covered

- Printing Output
- Comments
- Indentation
- Variables
- Data Types
- Type Casting
- type() Function
- Case Sensitivity
- Multiple Assignment
- Unpacking Variables
- String Concatenation
- Global Variables
- Local Variables
- global Keyword

## Requirements

- Python 3.x

## Run

```bash
python python_basics.py
```

## Output

```
Hello, World!
Five is greater than two.
Hello World
Python is Awesome
Excellent
```
